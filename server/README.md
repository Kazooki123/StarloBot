# StarloBot Server

Dedicated server for the Discord bot - StarloBot

## Introduction

Starlobot is a multi-purpose, fun, utility Discord bot made in Python, you can run `!customhelp` for more
commands of the bot.

Since it needs a way to host a dedicated server it uses **Rust** for it's security, reliability, performance and more.

It includes Shards, Nodes, Monitoring and a few more.

Starlobot Version: 1.1
